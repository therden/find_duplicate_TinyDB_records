# Find duplicates in a TinyDB database

Assuming that "duplicate entries" are defined as "records containing the same fields with the same field names and values (excepting their respective `doc_id` values)", this program demonstrates an approach to identifying duplicate records in a TinyDB database.

It was written in response to [this question](https://github.com/msiemens/tinydb/discussions/357) in the _Discussions_ section of [Marcus Siemens' TinyDB repo on Github](https://github.com/msiemens/tinydb).
